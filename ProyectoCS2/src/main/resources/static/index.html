Características Principales

Tecnología de vanguardia para la gestión médica moderna

Arquitectura Hexagonal

Diseño limpio y modular con separación clara de responsabilidades, facilitando el mantenimiento y la escalabilidad.

Bases de Datos Híbridas

MySQL para datos estructurados y MongoDB para historia clínica no estructurada, optimizando el rendimiento según el tipo de información.

Control de Acceso

Sistema granular de permisos por roles, asegurando que cada usuario acceda solo a la información correspondiente a su función.

API REST Completa

Endpoints RESTful documentados con Swagger, listos para ser consumidos por aplicaciones frontend o integraciones externas.

Reportes Dinámicos

Generación de reportes de facturación, pacientes y estadísticas con filtros avanzados y exportación de datos.

Interfaz Responsiva

Diseño adaptativo que funciona perfectamente en dispositivos móviles, tablets y computadoras de escritorio.

GET /api/patients Gestión de pacientes
POST /api/medical/orders Órdenes médicas
